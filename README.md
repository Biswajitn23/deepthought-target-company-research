# DeepThought — Target Company Research

## Overview
This project identifies and evaluates 25 ICP-qualified Federer companies in Hyderabad for DeepThought's Business Analytics Internship assignment.

## Components
- Part A: 25-company scored dataset (CSV)
- Part B: 1000-company scaling strategy
- Methodology documentation
- Hand-drawn system diagram

## Focus Areas
- Defence electronics
- Aerospace
- Specialty chemicals
- Precision engineering

## Key Framework
Companies are evaluated using C3–C8 scoring:
- Differentiation
- Decision-maker quality
- Sector tailwinds
- Growth signals
- Systems maturity
- Leadership succession

## Outcome
A structured pipeline for identifying high-quality MSME manufacturing companies in India.
